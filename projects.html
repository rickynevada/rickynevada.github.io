Personal

MicroMod

Wanted to use a different MCU for the Voydophone since the Delfino is over $20. Settled on one of the newest Piccolo MCU for $6 that has what is called a control law accelerator (CLA), a co-processor that will allow me to offload audio synthesize from the main processor. Decided to modularize Voydophone development for quicker evaluation and spinning of circuits.


ChargeMod

Voydophone needs a rechargeable battery circuit. Put reverse battery protection in after frying the last Voydophone...


3-axis gimbal with face tracking

A project I did for an independent study/research at UWB. Uses a commercial 3-axis gimbal that has an API exposed over UART to allow external control of the gimbal. I hooked up a PSoC 4 with BLE to route data from my Android phone, then I developed the Android App using a face tracking API to command position of the gimbal. Got it to work, but ran out of time in the quarter to implement any filtering or a control system to smooth out the movement.

Professional

First board I ever designed

Has a PIC32MZ, some boost circuits, external ADC, analog switch ICs, and multiplexer. I wrote the firmware for it and a Java GUI using JavaFX that controlled it over USB CDC.


Microelectromechanical System (MEMS) Inertial Measurement Unit (IMU) Calibration Fixture

Uses an STM32 and CAN bus to control calibration of 50 IMU boards. I wrote all the calibration firmware for the IMU boards (also STM32's) and the fixture MCU. Also developed a console based interface program in C# to communicate with the fixture over USB. I helped design and layout the PCB.



Programming and Test Fixture

I helped design and layout this board, and wrote all the firmware for the STM32 MCU, also developed a console based program to interface with the board in C# over USB.


Development Board

I designed and laid out the PCB. Mainly just a development board with the UARTs and other peripherals broken out. Also developed a Winforms GUI in C# to communicate with it over USB.


Data Logger

One my of first UIs. Wrote this program in Java using the Swing API to log data from four integrated circuit testers simultaneously. Eventually also converted a program written in assembly for the PIC24 on the tester (shown below) to C and added more capabilities. I did some simple IO routing on the MAX II CPLD too.
